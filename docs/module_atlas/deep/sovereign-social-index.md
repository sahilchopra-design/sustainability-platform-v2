## 7 В· Methodology Deep Dive

> вҡ пёҸ **GuideвҶ”code mismatch flag.** The guide's formula is `(HDIГ—0.30) + (InequalityГ—0.25) +
> (Labour RightsГ—0.25) + (Social Safety NetГ—0.20)` вҖ” a 4-dimension weighted blend sourced from
> UNDP/World Bank/ILO/IMF. **The code computes an unweighted mean of 8 dimensions**
> (`HDI, Gini Equality, Healthcare, Education, Labour Rights, Gender Equity, Press Freedom, Food
> Security`), and none of those 8 dimensions is itself sourced from UNDP/WB/ILO/IMF data вҖ” every
> one is generated by the same single formula conditioned only on income tier and GDP per capita,
> plus independent noise. There is no "Social Safety Net" dimension in the code at all. Sections
> below document what the code actually computes.

### 7.1 What the module computes

For 80 countries (`RAW_COUNTRIES`, hand-typed with real 2023-ish population/GDP figures), each of 8
social dimensions is generated by the **same shared formula**:

```
base = income==='High' ? 68 : income==='Upper-Mid' ? 48 : income==='Lower-Mid' ? 35 : 25
gdpFactor = clamp(gdp_pc/1200, 0, 20)
dim_k = clamp( base + gdpFactor + sr(idxГ—100 + kГ—17)Г—24 вҲ’ 8,  8, 98 )      // k = 0..7 (one per dimension)

composite = mean(dim_0 .. dim_7)                                            // unweighted, 1/8 each
```

Because every dimension shares the identical `base + gdpFactor` term and differs only by an
independent Вұ8-to-+16-point noise draw, **the 8 dimensions are highly correlated by construction** вҖ”
a country's HDI, Gini-equality, healthcare, education, labour, gender, press-freedom and
food-security scores will all cluster tightly around the same income-tier-implied centre, which is
not how real cross-country social indicators behave (e.g. the US has high HDI/healthcare-spend but
comparatively weak press-freedom-index performance relative to peers; that kind of divergence cannot
emerge from this formula beyond the shared Вұnoise band).

### 7.2 Parameterisation

| Constant | Value | Provenance |
|---|---|---|
| Income-tier `base` | High 68 / Upper-Mid 48 / Lower-Mid 35 / Low 25 | Synthetic вҖ” 4-tier World-Bank-style income classification is real, but the score offsets are hand-tuned, not fitted |
| `gdpFactor` | `clamp(gdp_pc/1200, 0, 20)` | Synthetic; caps at $24,000 GDP/capita (any richer country gets the same +20 boost) |
| Noise range | Вұ8 to +16 points (`sr()Г—24вҲ’8`) | Synthetic demo value |
| Clamp bounds | [8, 98] | Synthetic |
| Composite weights | 1/8 each (equal) | Contradicts guide's stated 0.30/0.25/0.25/0.20 |
| `bondYield` | `clamp(1.2+(100вҲ’composite)Г—0.06+sr()Г—1.5, 0.3, 14)` | Synthetic вҖ” a linear "social score вҶ’ yield" mapping, not an actual credit/spread model |
| `rating` | 15-tier scale (`AAA`вҖҰ`B`) from `ratingIdx=clamp(floor((100вҲ’composite)/7),0,14)` | Synthetic вҖ” implies social score alone determines credit rating, a significant methodological simplification |

### 7.3 Calculation walkthrough

1. **Dimension + composite generation** вҖ” as above, once per country at module load.
2. **Quarterly trend** вҖ” each dimension gets a 12-quarter path with a per-country/per-dimension
   drift term (`drift=(sr()вҲ’0.45)Г—2.5`, multiplied by `(qiвҲ’6)` so it fans out from the current
   quarter), clamped to [5,99].
3. **SDG scores** вҖ” each of the 17 UN SDGs (8 flagged `social:true` вҖ” SDGs 1,2,3,4,5,8,10,16) gets
   a score `= clamp(compositeГ—0.7 + sr()Г—35, 5, 98)` and a `target2030` and a boolean `bondAlign`
   flag (`sr()>0.5`) вҖ” again, everything traces back to the same composite plus fresh noise, not to
   distinct SDG-indicator data.
4. **Sovereign bond linkage** вҖ” `bondYield` and `rating` are both monotonic, deterministic functions
   of `composite` (plus small noise on yield only) вҖ” a country's synthetic yield and rating are
   *fully determined* by its synthetic social score, which is a real modelling relationship (better
   social composite вҮ’ tighter yield вҮ’ higher rating tier) even though the underlying inputs are
   synthetic.
5. **Peer group** вҖ” categorical bucket (`Leader вүҘ80`, `Strong вүҘ60`, `Moderate вүҘ40`, `Weak <40`) off
   `composite`.
6. **Cross-cuts** вҖ” regional averages, income-group breakdowns, quartile rankings, a Pearson
   correlation matrix (`corrMatrix`) computed across the 8 dimensions **within this same synthetic
   dataset** вҖ” meaningful as a demonstration of the shared-driver correlation structure described in
   В§7.1, not as evidence about real-world social-indicator correlation.
7. **Portfolio** вҖ” 25 synthetic sovereign bond holdings with weight, adjusted yield
   (`c.bondYieldГ—(1+(50вҲ’composite)/200)`), normalised to 100%.

### 7.4 Worked example вҖ” Denmark (index 0)

`income='High'`, `gdp_pc=$67,800` вҶ’ `base=68`, `gdpFactor=clamp(67800/1200,0,20)=20` (capped).

| Dimension | `sr(0Г—100+kГ—17)` | Result |
|---|---|---|
| HDI | sr(0) | 97.0 |
| Gini Equality | sr(17) | 83.1 |
| Healthcare | sr(34) | 84.2 |
| Education | sr(51) | 86.6 |
| Labour | sr(68) | 83.6 |
| Gender | sr(85) | 90.0 |
| Press | sr(102) | 98.0 (clamp ceiling) |
| Food | sr(119) | 82.7 |
| **Composite** | mean | **88.15** |

Downstream: `bondYield = clamp(1.2+(100вҲ’88.15)Г—0.06+sr(600)Г—1.5, 0.3, 14) вүҲ 2.98%`;
`ratingIdx = floor((100вҲ’88.15)/7) = 1` вҶ’ **rating "AA+"**. This lands close to Denmark's real AAA
sovereign rating вҖ” a coincidence of the formula's calibration for very-high-income countries, not
evidence the model tracks real creditworthiness drivers.

### 7.5 Companion analytics

- **Dimension correlation matrix** вҖ” Pearson correlation across the 8 dimensions computed from the
  80-country synthetic panel; will show artificially high positive correlation (see В§7.1).
- **SDG alignment summary** вҖ” counts of countries meeting bond-alignment flags per social SDG,
  purely cosmetic given `bondAlign` is an independent coin-flip.
- **Portfolio tab** вҖ” 25-holding synthetic sovereign social-bond portfolio with normalised weights
  and a composite-adjusted yield.

### 7.6 Data provenance & limitations

- **Fully synthetic scores.** No UNDP HDI, World Bank Gini, ILO EPLEX, or IMF Fiscal Monitor data is
  ingested despite being named in the guide вҖ” `RAW_COUNTRIES`' population/GDP figures are real-ish
  hand-typed context, but every social dimension score is generated from income tier + noise only.
- The 8-dimension unweighted mean structurally cannot reproduce the guide's 4-dimension weighted
  formula, and the "Social Safety Net" dimension named in the guide does not exist in code at all.
- `bondYield`/`rating` being deterministic functions of the *social* composite alone (with no debt,
  fiscal, or macro input) is a stylised toy relationship вҖ” real sovereign ratings weight fiscal
  metrics, external liquidity, and political risk far more heavily than social indicators per S&P/
  Moody's published criteria.
- Same-seed correlation structure means any statistical analysis run against this dataset (e.g. the
  in-page correlation matrix) will overstate real-world indicator correlation.

**Framework alignment:** UNDP HDI / World Bank Gini / ILO EPLEX / IMF Fiscal Monitor are named
guide sources but not implemented; UN SDG framework (17 goals, correct social-SDG flagging of
1/2/3/4/5/8/10/16) is reproduced structurally though scores are synthetic. See sovereign-esg-scorer
and sovereign-esg-scorer's В§8 for the indicator-normalisation model this module would need to
genuinely implement its own guide's formula.
